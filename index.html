Objectivo:

Criar um modelo de curriculun vitae e/ou página web de portfolio para que você ou seu possivel empregador possam acessar, visualizar e baixar seus dados a partir de qualquer lugar a qualquer momento. Evitando perdas de dados dispensaveis, o portfolio pode e deve ser actualizado, sempre que os seus dados pessoais e habilidades assim o requererem.

Como funciona?

Ao clicar em quero um curriculo ou portfolio, que abrirá a nossa lista de contactos (email, whatsapp, facebook), na qual selecionarás a opção em que efectuarás a troca de dados. Para a criação do CV ser-te-á perguntado qual o número do modelo que desejas, em seguida responderás aos formulários para o preenchimento do mesmo. Os dados de são submetidos ao site a meia noite, de forma gratuita, será enviado um link para o seu contacto de modo que possas acessa-lo sempre que necessario. Contudo de modo a evitar a usurpação de dados ou até mesmo que alguem altere os dados de seu portfolio, ser-te-á dado um código numérico que usarás sempre que precisares modificar o portfolio.

Os portfolios são compostos por uma pagina guia vinculada ao seu CV, Copia do BI, Fotos, Cartas de Recomendação, conforme desejar. Sempre que for necessario poderás fazer o download de seus documentos oad está vinculado aos dados(compactados).
